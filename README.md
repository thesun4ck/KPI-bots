# 📊 KPI Tracker Bot

Telegram-бот для предпринимателей малого бизнеса. Помогает отслеживать ключевые метрики бизнеса прямо в Telegram — без Excel и сложных инструментов.

---

## Как это работает

Каждый вечер бот сам пишет и спрашивает цифры за день. Ты отвечаешь на несколько вопросов, данные копятся в базе. Раз в неделю бот присылает сводный отчёт с графиками и сигнализирует если что-то пошло не так.

---

## Возможности

- 📋 **Ежедневный ввод метрик** — выручка, клиенты, средний чек и другие показатели под твой тип бизнеса
- 📈 **Отчёты с графиками** — за неделю, за месяц, сравнение периодов
- ⚠️ **Алерты** — бот сигнализирует если метрика упала больше чем на 30% от среднего
- 🎯 **Цели** — устанавливай план на неделю/месяц и следи за выполнением
- 📤 **Экспорт** — выгрузка всех данных в CSV
- 🔐 **Админ-панель** — статистика бота, список пользователей, бэкап БД

---

## Типы бизнеса

При регистрации бот автоматически настраивает метрики под твой бизнес:

| Тип | Метрики |
|---|---|
| 🛒 Розничный магазин | Выручка, чеки, средний чек |
| ☕ Кафе / ресторан | Выручка, чеки, средний чек, списания |
| 💻 Онлайн-бизнес | Заявки, продажи, выручка, конверсия |
| 🛠 Услуги | Выручка, записи, явка, средний чек |
| 📦 Другое | Выручка, клиенты |

## Запуск локально

**1. Клонируй репозиторий**
```bash
git clone https://github.com/твой_ник/kpi_tracker_bot.git
cd kpi_tracker_bot
```

**2. Установи зависимости**
```bash
pip install -r requirements.txt
```

**3. Создай `.env` файл**
```env
BOT_TOKEN=твой_токен_от_BotFather
ADMIN_IDS=твой_telegram_id
```

**4. Запусти бота**
```bash
python kpi_tracker_bot/bot/main.py
```

---

## Деплой на Railway

1. Подключи репозиторий в [Railway](https://railway.app)
2. Добавь переменные окружения в разделе **Variables**:
   - `BOT_TOKEN`
   - `ADMIN_IDS`
3. Railway автоматически подхватит `Procfile` и запустит бота
